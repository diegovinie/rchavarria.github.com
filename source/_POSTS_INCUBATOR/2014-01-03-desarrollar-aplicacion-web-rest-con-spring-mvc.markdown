2014-01-03-desarrollar-aplicacion-web-rest-con-spring-mvc

Spring MVC es un framework dentro de todo el ecosistema de Spring para el
desarrollo de aplicaciones web usando una arquitectura MVC
(Modelo-Vista-Controlador). 

En principio, el framework está pensado para que la *Vista* sean páginas
JSP, pero no tiene porqué ser así. ¿Qué pasa si el resultado de una
petición al servidor se renderiza como unos datos en formato JSON? Pues
resulta que tenemos una aplicación web que se comporta como si fuera
un servicio web REST.

Y a partir de aquí.... ¿cómo desarollar un servicio web rest con spring mvc?

Definir las url's, los códigos de retorno, empezar por controladores mvc,
configurar la app, ...

Comenzar la aplicación de prueba a partir de este tutorial:
http://spring.io/guides/tutorials/rest/
